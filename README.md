# Reverse Engineering Learning Journey

Repositori ini berfungsi sebagai catatan dan jurnal belajar pribadi yang berfokus pada dasar-dasar **Reverse Engineering (RE)** untuk kebutuhan cybersecurity. Dokumen di dalamnya dirancang untuk mendokumentasikan perkembangan pemahaman dari konsep paling fundamental hingga praktik analisis biner secara langsung.

Tujuan utama dari proyek ini adalah mengubah materi teknis yang kompleks menjadi catatan yang lebih mudah dipahami di masa mendatang. Pembahasan di dalam repositori ini mencakup teori dasar arsitektur sistem, bahasa assembly, hingga struktur file biner.

Selain teori, repositori ini juga memuat dokumentasi praktik langsung untuk menguji pemahaman konsep yang telah dipelajari. Praktik tersebut mencakup dua pendekatan utama dalam analisis perangkat lunak:
* **Analisis Statis:** Membedah komponen biner tanpa mengeksekusi programnya.
* **Analisis Dinamis:** Mengamati perilaku program secara real-time saat dijalankan.

Proses pengujian dilakukan menggunakan program sampel orisinal yang dibuat khusus untuk simulasi tantangan keamanan biner. Beberapa perkakas standar industri seperti disassembler dan debugger digunakan sepanjang proses belajar ini untuk membantu memahami bagaimana sebuah kode mesin bekerja di balik layar.
